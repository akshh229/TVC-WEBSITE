==============================================
THE VOYAGE CLUB — ASSETS FOLDER
==============================================

Replace IMAGE_LINK_HERE placeholders throughout
the HTML files with actual image paths like:

  src="assets/images/hero-photo.jpg"
  src="assets/images/team/president.jpg"
  src="assets/images/events/summit-2026.jpg"

Recommended image structure:
  assets/images/
    ├── logo.png
    ├── hero/
    ├── team/
    ├── events/
    ├── gallery/
    └── sponsors/
